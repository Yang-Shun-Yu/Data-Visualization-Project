# Data Visualization using D3.js

## Overview

Transform data into meaningful graphical forms using D3.js and web technologies. D3 is a JavaScript library for visualizing data with HTML, SVG, and CSS. 

## D3.js graph gallery

- Distribution
  - Histogram
- Correlation
  - Correlogram
  - Heatmap   
- Evolution
  - Area
  - Stacked area
